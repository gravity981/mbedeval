# Atom IO editor

### Cheatsheet
* ``ctrl + shift + p`` opens command search mask
* ``ctrl + ,`` settings
* ``ctrl + t`` to open  file
* ``ctrl + shift + m`` to open markdown preview

### Additional packages
* language-batchfile
* platformio-ide-terminal
* switch-header-source
